JavaScript, también conocido como Moka, LiveScript, JScript, ECMAScript es uno de los lenguajes de programación más popular en el mundo del desarrollo web. Prácticamente, cada computadora moderna tiene un intérprete de **JavaScript** instalado. La popularidad de JavaScript se debe principalmente al papel que juega como lenguaje de guiones en la red.

Sin embargo, a pesar de su popularidad, pocos saben que JavaScript es un lenguaje de programación propósito general bastante dinámico. ¿A qué se debe esto? ¿Acaso no se comprende o algo así?

### El nombre de JavaScript

El prefijo _Java_ sugiere que __JavaScript__ está relacionado de alguna manera con el lenguaje de programación Java de Oracle. Tal parece que el nombre fue elegido intencionalmente para causar confusión, y de la confusión vienen los malos entendidos. __JavaScript no es Java interpretado__. Java es Java, un lenguaje interpretado precompilado. JavaScript es un lenguaje de programación diferente.

JavaScript tiene una sintaxis similar a Java, de la misma manera que Java tiene una sintaxis similar a _C++_. Pero esto no significa que Java sea un subconjunto de C++.

JavaScript no fue desarrollado por Sun Microsystems, el creador de Java. JavaScript fue desarrollado por Netsacape. Originalmente se llamaba _LiveScript_, pero ese nombre no confundía a nadie[^1].
